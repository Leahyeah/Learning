# 05 现金流量表：四步分析法，教你准确判断企业究竟能活多久（实战篇）

第四节  现金流量表：四步分析法，教你准确判断企业究竟能活多久（实战篇）

行家带你读财报，升职赚钱有门道。

你好，我是对外经济贸易大学会计学教授张新民。

欢迎你来到我的财报分析课堂。

这一节，我们要看看企业的钱是怎么来的，又是怎么用的。

这就要看现金流量表了。

我先问同学们一个问题啊：你认为：一个企业的生存，是靠利润还是靠现金流量？

你可能会觉得，嗯，公司大都是以盈利为目的，利润当然重要。

但是，你可能又想起一句名言：现金为王！

你可能还听过，海尔集团董事局主席张瑞敏说过这么一句话：

一个企业没有利润不一定破产清算，但如果没有现金流量就一定破产清算！

现金为王和张瑞敏的说法是广为流传的，这两句话实质是在强调现金流量的极端重要性。

现在我回答前面的问题，一个企业的生存，如果只是靠利润，或者只是靠现金流量，都是不可能的。

比较恰当的回答是：持续发展的企业，既要靠利润，也要靠现金流量；

在短时间内，对于一个企业的生存而言，现金流量比利润更重要！

所以，看完了利润表，我们赶紧要看一下现金流量表。

如果你对现金流量表完全没有概念，建议你先听听前面的预习课程，再学习下面的内容。

这节课，我将带你一起分析现金流量表。重点要与你交流两个问题：

首先，从一个我提前预测到公司很快会垮台的案例中，体会现金流量表的价值。

然后，我要用我独创的四步分析法，告诉你现金流量表怎么看？什么样的现金流量结构能让企业发展的更持久？

一、乐视网解密

现在，我就先跟大家分享一个我提前预测到的很快垮台的公司的案例。

当年被称为“创业板第一股”的明星企业，这家企业的创始人通过给投资人讲“打造生态”的故事，把公司的市值推到了超上千亿元的位置。

当时很多人都说他在创造了奇迹，正在上升时期的时候，我仔细分析了这家企业2016年的报表。

发现这个企业问题非常大，如果企业不做改变，肯定过不了多长时间就会垮台，而这些问题不懂财报的吃瓜群众压根就没有察觉。

你可能已经猜到了，这家企业就是乐视网。

而现在，我们回过头来看，如果早点听了我的课程，很多人只要对这个企业的财务报表稍加注意，就会从报表中，很早就能看到这个企业光鲜亮丽的背后所隐藏的巨大风险。

你会问了：张老师，你是怎么看出来的？

我要告诉你，非常简单。这个企业的现金流量表早就对我们进行了预警。

听我慢慢讲给你听。

我先把乐视网2016-2018年三年的合并现金流量表展示给你们。

实际上，我是在2017年上半年开始注意到这个企业2016年现金流量表中所隐藏的巨大风险的。

由于这个企业在2016年到2018年期间，每年的现金流量表的结构非常相似，我就用三年的数据来给大家做说明。

现在，我们拿到了乐视网的现金流量表。

首先，请大家考察企业经营活动产生的现金流量的情况。

从报表的情况来看，企业2016年至2018年三年间，不管利润表的核心利润如何，经营活动产生的现金流量净额都是负数！

2016年大约是-11亿元，2017年大约是-26亿元，2018年大约是-11亿元！

这就是说，整个经营活动长期处于货币资金入不敷出的状态！

这说明，企业经营活动的造血功能长期不足！

这就相当于一个已经参加工作的人，他每年挣得钱，还不够满足自己的日常开支呢！

想想看，这种货币资金长期入不敷出的缺口弥补，应该由谁来完成？

如果是你自己个人，你要么去消耗原来存下来的钱，要么跟父母要，要么去四处借钱。

企业怎么办呢？我们先放在这里。

然后，请大家考察企业投资活动现金流出量的情况。

在企业投资活动现金流出量里面，连续三年购建固定资产、无形资产和其他长期资产支付的现金都有一定规模，2016年约为55亿元，2017年约为26亿元，2018年约为2亿元。

呈现出逐年下降的态势。

这么多的现金流出量买了什么？

我告诉大家，主要是购买了用于企业经营的各种无形资产！

就是各种著作权之类的无形资产。

这种现金流量规模越来越小说明，在这三年里，企业在通过购买各类权力来支持业务发展的底气越来越不足。

讲到这里了，请大家再关注一件事情：企业经营活动产生的现金流量净额与投资活动产生的现金流量净额之和是多少？

很清楚了：在这三年里，企业的经营活动没有能力带来钱，投资活动还要大量花钱。

两项加在一起，2016年的资金缺口是107亿元，2017年资金缺口是46亿元，2018年资金缺口小了点儿，但仍然有缺口11亿元。

这就相当于一个个体，自己的每年挣的工资还不够支付自己的日常开支，还要买房买车，这不是资金的缺口越来越大吗？

如果是你自己身边的朋友有这种情况，你会怎样看他？

你如果是这个企业的领导，你会着急吗？

你如果是这个企业的领导，你要到哪里去找钱，来维持你这个企业的发展呢？

你在与别人交流的时候，代表这样的企业，你会有底气吗？

看到这里，大家是不是感觉到了乐视网背后可能存在的危险？

什么危险？资金流断裂的危险！

当然，在我们看到的乐视网这三年中，它的资金流是没有断裂的。

那它是靠什么维持资金链呢？

接下来，第三个关键点，请大家看看企业筹资活动产生的现金流量的规模、结构和支持方向。

我们在前面的分析中有了一个基本的感觉：乐视网的经营活动资金有很大缺口，投资活动有很大资金需求。

这些都需要筹资来支持。

看一下，企业的筹资活动现金流量是怎样表现的：还真不容易！这三年，每年企业都是靠筹资在苦苦支撑。

2016年，主要靠发行股票筹资111亿元，然后是借款，最后整个筹资活动贡献了约95亿元的资金；

2017年，主要靠子公司的非控制性股东入资31亿元，然后是借款，最后整个筹资活动贡献了约39亿元的资金；

2018年，主要靠子公司的非控制性股东入资7亿元，然后是借款，最后整个筹资活动贡献了约9亿元的资金。

可以这样说，企业的筹资活动现金流量类似于人的输血；经营活动现金流量类似于人的造血。

经营活动现金流量为负数，就是企业自身造血能力不足的表现。

你认为纯粹靠输血、自身造血能力严重不足的企业能走远吗？

社会上哪里有那么多的钱让你去烧？让你去挥霍？

听到这里，你是不是也有一种跃跃欲试的感觉，想去看一下你所关心的企业，它的生存是靠造血还是输血？

二、现金流量表的“四步分析法”

好了。

对于企业现金流量表的分析，你只要按照我总结的“四步分析法”去做就行了。

对于大多数人来说，其实完全没有必要对照现金流量表一项一项的看。

这个现金流量表的【四步分析法】，你就能把握企业现金流量的关系，进而对企业的持续发展做出判断。

第一步，考察企业经营活动产生现金流量的充分性。

怎样才算经营活动现金净流量充分呢？当然是与企业的核心利润比。

我们在上一节课讲了，与营业收入有关的利润既不是营业利润，也不是利润总额，更不是净利润，而是核心利润。

因此，经营活动产生的现金流量净额一定是与核心利润去比较。

怎么比较呢？

我们可以用一个比率，叫核心利润获现率，来衡量这方面的问题。

核心利润获现率 = 经营活动产生的现金流量净额 / 核心利润

用这个比率，既可以衡量企业核心利润的含金量，也可以在很大程度上衡量企业经营活动现金流量的充分性。

那么，这个比率多大比较恰当呢？

对于一般性的固定资产无形资产占资产比重不大的企业，我的经验是，这个数据在1.2-1.5之间是比较理想的。

这就是说，理想的经营活动产生的现金流量是要达到核心利润的1.2-1.5倍的。

企业既要追求利润，更要追求现金流量。

从企业现金流转的情况来看，经营活动产生的现金流量展示的是企业的造血能力！

而企业从股东那里拿到的现金属于“输血”。

第二步，考察企业投资活动现金流出量的扩张含义。

这就是说，投资活动现金流出量的结构，会反映企业在年度内想不想扩张、以及怎样扩张的情况。

你想啊，如果一家企业感觉自己的发展潜力比较大，还想扩张，他肯定会买设备、买入一些必要的专利啊，专有技术啊，

甚至于可能还要买地，来促进公司业务的发展。

我们现在就来看看这些活动在现金流量表上是怎样体现的。

就拿乐视网来说，这个公司2016年至2018年三年的投资活动现金流出量的两个主要项目为：

一个是“购建固定资产、无形资产和其他长期资产支付的现金”，如果你买设备、建房屋、买地等花的钱，都计入这个项目；

另一个是“投资支付的现金”，这个投资的范围是很广的，既有对外的长期给子公司的投资，也有各种各样的投资，比如购买其他公司的股票、债券等等。

对于大多数企业而言，投资活动产生的现金流出量，表现出了不同的扩张含义。

从乐视网的现金流量表来看，在这三年里，“购建固定资产、无形资产和其他长期资产支付的现金”成了投资活动现金流出量的主流，各种投资并不多。

这意味着，公司的投资活动主要集中在支持企业经营活动的固定资产和无形资产的购建上。

可惜的是，企业的这种支持企业经营活动的资产增加并没有带来企业经营活动的真正发展。

总结一下，投资活动现金流出量与企业扩张之间的关系。

● 如果购建固定资产、无形资产支付的现金规模较大，则一般说明企业还想通过扩大固定资产和无形资产的规模来强化自己的生产经营能力，还想发展。比如说，你们公司的办公室扩大一倍，而且还增加了很多工位和电脑，那一开就是企业想要扩大发展。

● 如果购建固定资产、无形资产支付的现金规模很小，说明企业是在原有基础设施支持的条件下，按照惯性发展。

● 如果企业的投资支付现金比较多，则说明企业可能在对外投资扩张上有所作为，或者在进行频繁理财。比如说，你们公司现金流充足，还有点闲钱，会买点股票，或者到其他公司买点股份，这时候就会支付大量现金。说明公司对外投资的扩张。

第三步，考察企业筹资活动产生的现金流量的规模、结构和支持方向。

我们就是要看看，谁在给企业钱？企业的钱用在了哪些方面？

前面已经讲过，筹资活动现金流量展示的是企业被输血的情况。

我们看一下，谁在支持乐视网所需要的资金。

乐视网来2016年至2018年三年的筹资活动现金流入量的情况为：

注：企业的附注显示，（1）2018年“收到其他与筹资活动有关的现金”为“取得非金融机构借款”，这实质应该属于“取得借款收到的现金”；（2）2017年“收到其他与筹资活动有关的现金”中包含了一项“处置子公司部分股权”获得的2,301,760,000元款项，这实质应该归属于“收回投资收到的现金”

从乐视网的报表来看，向企业提供资金的主要是这几个力量：

第一，上市公司的股东，如2016年，乐视网发行股票获得约111亿元的股东入资，不知道这些当年购买乐视网发行的股票的股民是赚了还是亏了；

第二，子公司的非控制性股东，也叫少数股东，如2017年和2018年，子公司的少数股东分别向子公司入资31.11亿元和7.3亿元。你看乐视网多能忽悠股东呀！

第三，各类金融机构。在2016年至2018年间，金融机构向乐视网提供的资金三年累计达到了130亿元。

而企业获得的股东和金融机构所提供的资金，一部分用在了补充经营活动现金缺口，一部分用在了补充购建固定资产无形资产方面的资金需求。

正是股东和金融机构不遗余力地支持乐视网，才使得这个企业烧钱的底气很足。

总结一下：

我们看筹资活动现金流量，主要看四个方面：

第一，看看股东给钱了吗？

第二，看看各种金融机构给企业钱了吗？

第三，看看企业发行债券筹资了吗（乐视网在2016至2018年间没有发行债券）？

第四，看看企业筹到的钱花在了什么地方？

企业筹资获得的钱主要用在五个方面：

第一，补充经营现金流量的入不敷出，就像乐视网；

第二，企业要做大，要继续购买固定资产，什么机器呀、设备啊，或者买无形资产了，比如商标呀、专利呀等，就像乐视网；

第三，收购公司或者搞其他投资了，如乐视网的2016年；

第四，用来还债了。有的时候，企业要借新债还旧债；

第五，本来没有需求，但还是要借钱。这种情况最不应该，完全是企业自己在那儿借着钱玩呢。这种情况筹集到的钱，会增加货币资金的规模。

第四步，体会什么样的现金流量结构能让企业发展的更持久？

想想看。你投资了两个公司。

你投资的第一家公司：A公司，这家公司经过一段时间的管理和经营，

迅速形成了市场规模，产品的市场竞争力很好。

由于竞争优势明显，A公司大量采取了预收货款的方式销售。

不仅如此，A公司对供应商也有很好的商业信用，

供应商可以在A公司将货物卖出后再收取货款。

这就是一个上下游“两头吃”能力极强的企业。

这个企业的核心利润获现率很高，达到了2倍以上。

你投资的另外一家B公司呢，经过一段时间的管理和经营，

虽然形成了一定的市场规模，但产品的市场竞争力较差。

由于没有什么竞争优势，B公司只能大量采取赊销的方式销售。

不仅如此，B公司在供应商那里也不能很好的利用商业信用，

供应商在供货前就要求B公司将货款全部支付后才能发出货物。

真是冰火两重天呀！

B公司够惨的吧：销售回款回不来，购货付款要提前支付，

想想看，这样的公司经营活动发生的现金亏空由谁来弥补？

当然是由股东来帮忙解决了。就算是公司去借钱，恐怕也要由股东来担保。

这种经营活动现金流量能力差的公司能走多远呢？

这要看股东的钱能让他烧多长时间。

B公司由于市场竞争力较差而导致连年亏损，

经营活动产生的现金流量净额持续为负数。

你作为股东，要不断给B公司寻找资金去支持他的经营活动。

别以为我讲的这样的公司是笑话。

实际上，这样的公司可能就在你的身边。

实际上，我们每一个人也是这样：如果你在一个单位打工，

你的日常工资奖金等就是你的经营活动现金流入量，

你的日常开支包括吃饭、交通、需自己负担的房租、穿衣、学习等费用支出就是你的经营活动现金流出量。

如果你的日常工资不能维持你的生活的话，一定会出现亏空。

这时，你会怎么办呢？

第一，找父母要，这类似于股东持续向企业入资；第二，找别人借，这类似于企业借款；第三，用信用卡买东西（采取推迟付款的方式）等。

这样的伸手要求的感觉是不是太不容易了？

所以，大家也要像管理一家公司一样，好好经营自己的生活。

注意收入和支出的情况，让日子过得越来越好！

到这里，我们就可以得出一个结论：

经营活动现金流量足够充分的企业可能发展的更持久。

这就是说，造血能力强的企业才能够发展的更持久！

请大家一定要记牢。

在这一节，我用四步分析法教你如何看现金流量表。

总结一下：

● 经营活动现金流量看净额，重点考察企业经营活动的造血能力；

● 投资活动现金流量看流出量，重点考察企业的扩张意图；

● 筹资活动现金流量看流入量，重点考察企业被输血的情况。

现金流量表的核心是经营活动现金流量的充分性。

经营活动现金流量能力突出的企业能够走得更远；

投资活动流出量应重点关注构建固定资产、无形资产和其他长期资产支出

体现了企业经营扩张的动作，直接强化企业未来经营能力；

筹资活动现金流量重点关注流入量，

流入量的结构在一定程度上代表了企业所能够吸引资源的途径。

在结束本节课之前，我想再跟大家交流一下，由2020年初新冠肺炎导致的中小企业临时性财务困境和应对问题。

由于部分城市封城、交通阻断、2020年春季开工的不断推迟以及城际间交通与物流能力的大幅度下降，致使一些企业迅速陷入了财务困境。

第一，市场萎缩，经营现金流入量大幅减少

在商品和服务市场方面，由于国家上下全力抗击新冠肺炎而导致了市场方面与抗击新冠肺炎有关的医疗卫生产品短缺，大量传统产品的市场出现萎缩。

对于受市场萎缩影响企业而言，其至关重要的经营活动产生的现金流入量首先出现下降。而这种下降在一段时间内还会持续下去。

与此同时，在新冠肺炎以前所产生的债权（企业的应收账款等）的可回收性也在变差——一些债务企业由于担心自己未来的市场会持续萎缩而推迟对商业负债的清偿，从而导致债权企业的经营活动产生的现金流入量进一步恶化。

第二，刚性支出增加

即使企业出现萎缩的经营活动可以在一段时间依靠已有的库存和推迟支付员工工资来进行，企业的某些活动如销售活动等不得不推迟进行，企业的一些刚性支出不但不能缩小，反而可能加大：经营场所租金可能需要尽快支付、企业与疫情防疫有关的支出必然增加、企业还可能支付一些赈灾捐赠等。

因此，本来已经在经营活动现金流入量陷入困境的企业，可能还要面临某些刚性支出进一步增加的情况。这对企业的现金流量管理无异于雪上加霜。

第三，筹资渠道受阻，筹资规模受限

由于对新型肺炎结束的时间难以判断，企业的债务筹资渠道和规模在一定程度上会受到限制。尽管有些金融机构推出了惠及债务筹资方的信贷政策，但对于很多企业而言，难以从金融机构获得增量贷款以应对新肺炎疫情所带来的财务困境是他们必须面对的现实。

困境和危机来了，怎么应对呢？

我的建议如下：

第一，现金流量管理聚焦生存，暂缓发展。

常规情况下的企业现金流量管理会兼顾企业的经营与发展。

在抗击新冠肺炎的环境下，企业现金流量管理的着眼点就要从兼顾经营与扩张或发展调整为着力解决企业生存方面。

具体而言，在现金流量支出的安排上，企业应首先满足经营活动现金流量的支出。在现金流量相对紧张的条件下，企业应该尽量推迟安排扩张性支出。这样安排，虽然可能推迟实施具有战略意义的企业扩张或发展项目，但却能够在一定程度上解决企业的生存问题。

在面临生存与发展抉择问题的时候，企业只有先生存下来，才能谋求后续的发展。

第二，成本管理聚焦酌量性成本。

学过管理会计的同学知道，在企业的成本中，按照与企业当前管理层的管理决策的关联性，可以区分为酌量性成本和约束性成本。其中，在抗击新冠肺炎过程中的酌量性成本是指那些直接取决于企业当前管理层的决策的成本，比如与选择哪种防疫器材与装备、采用怎样的抗击新肺炎的人力资源工作安排、与抗击新肺炎有关的加班津贴和补贴，以及采用何种宣传媒介、宣传强度等有关的成本或支出。

而约束性成本则是指那些与以前管理层决策有关或与当前管理层的决策不能控制的因素有关、但在现在或未来支付、与当前管理层的决策无关的成本或支出，如抗击非典过程中必须支付的企业水电费、企业发生的与其被强制性征用设施有关的费用、由企业以前决策决定的本期及未来需要支付的经营场所租金以及各种购买支出等。

显然，抗击新冠肺炎期间的成本管理应该聚焦在酌量性成本或者支出上。

第三，筹资结构从债务融资到债务融资与股权融资的合理安排。

在困境中，如果企业没有存量货币资金可供消耗，就要考虑筹资了。

在筹资方面，企业当然要继续争取金融机构的贷款。在债务筹资收到限制而企业必须筹资的时候，企业的筹资对象就要考虑从“自家人”那里获得了。

这里的“自家人”是指企业的股东和员工。对于企业而言，既可以在关键时刻向自己的员工借款以度过难关，也可以向自己的股东借款。当然，股东们也可以协商一致，按照一定的约定向企业注入增量资本；或者股东们在借给企业部分款项的同时，还向企业注入增量资本。

困境中的现金流量管理的要诀是：融资规模优于融资成本。只有生存了，才能发展。挺过难关，现金流问题一定有办法解决。
