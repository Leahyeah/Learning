# 学习卡｜增长占用现金

- 日期：2026-07-23
- 主题：增长占用现金（growth consumes cash）
- 边界：用于训练财报与估值分析思路；不构成投资结论。
- 核心问题：收入增长时，为什么经营现金流可能没有同步增长，甚至转弱？

## 1. 技术

- **具体问题**：怎样把“利润增长”和“现金增长”分开观察？
- **可验证事实 / 来源 [共识]**：IAS 7 要求将现金流量按经营、投资、筹资活动分类；经营活动现金流用于反映经营是否产生足够现金。[IFRS Foundation：IAS 7](https://www.ifrs.org/issued-standards/list-of-standards/ias-7-statement-of-cash-flows.html)
- **传导**：先以经营现金流为观察出口；再将利润调整项与营运资本变动拆开，进入产品成交和回款的场景。

## 2. 产品 / 场景

- **具体问题**：新增销量发生时，企业是否须先采购、生产或备货，才可能确认收入？
- **可验证事实 / 来源 [共识]**：存货售出后，其相关成本转入利润表的营业成本；未售出的存货仍是流动资产。[课程材料：存货周转](https://github.com/Leahyeah/Learning/blob/main/00_%E8%B4%A2%E6%8A%A5%E5%88%86%E6%9E%90%E8%AF%BE%E7%A8%8B/%E8%B4%A2%E6%8A%A5%E8%AF%BE%E7%A8%8B%E8%B5%84%E6%96%99Markdown%E5%B9%B3%E9%93%BA/171-04%E8%B4%A2%E6%8A%A5%E6%8E%98%E9%87%91-14%20%E5%AD%98%E8%B4%A7%E5%91%A8%E8%BD%AC%EF%BC%9A%E8%B4%A7%E5%A4%AA%E5%A4%9A%E8%BF%98%E4%B8%8D%E7%9B%88%E5%88%A9%EF%BC%9F%E4%B8%89%E6%AD%A5%E6%8F%AD%E7%A7%98%E5%AD%98%E8%B4%A7%E5%8F%98%E7%8E%B0%E9%87%91%E6%B5%81%E7%9A%84%E8%83%BD%E5%8A%9B.md)
- **传导**：若销量增长伴随存货或应收款先增加，现金可能先留在经营资产中；这要求继续看上下游的结算条件。

## 3. 产业

- **具体问题**：产业链中的账期、备货和预付款规则，如何决定增长需要多少额外经营资金？
- **可验证事实 / 来源 [共识]**：营运资本相关的典型经营项目包括存货、预付款项、应收账款，以及应付账款和预收款项；课程同时提示，流动资产/负债还混有投资和借款项目，不能直接等同于经营。[课程材料：营运资本概述](https://github.com/Leahyeah/Learning/blob/main/00_%E8%B4%A2%E6%8A%A5%E5%88%86%E6%9E%90%E8%AF%BE%E7%A8%8B/03-%E5%BC%A0%E6%96%B0%E6%B0%91%E3%80%8A%E8%B4%A2%E5%8A%A1%E6%8A%A5%E8%A1%A8%E5%88%86%E6%9E%90%E6%A1%88%E4%BE%8B%E9%9B%86%E9%94%A6%E3%80%8B90%E9%9B%86%E8%A7%86%E9%A2%91%E8%AF%BE%E8%BD%AC%E5%86%99-%E5%B9%B3%E9%93%BA/038-%E5%AE%8C%E7%BB%93-%E5%BC%A0%E6%96%B0%E6%B0%91%E8%B4%A2%E5%8A%A1%E6%8A%A5%E8%A1%A8%E5%88%86%E6%9E%90%E6%A1%88%E4%BE%8B%E9%9B%86%E9%94%A6-07-%E4%B8%93%E9%A2%98%E4%B8%83-%E8%90%A5%E8%BF%90%E8%B5%84%E6%9C%AC%E7%AE%A1%E7%90%86%E5%88%86%E6%9E%90-%E7%AC%AC%E4%B8%80%E8%8A%82-%E8%90%A5%E8%BF%90%E8%B5%84%E6%9C%AC%E7%9A%84%E5%8F%91%E5%B1%95%E5%8E%86%E7%A8%8B%E4%B8%8E%E6%A6%82%E8%BF%B0.md)
- **传导**：将报表项目按经营性与非经营性重分类后，才能判断收入增长究竟提高了应收/存货占用，还是由应付/预收提供了资金。

## 4. 资本

- **具体问题**：增长造成的现金缺口，会如何改变企业对外部资本的依赖？
- **可验证事实 / 来源 [共识]**：现金流量表中的筹资活动，是导致投入资本和借款规模或构成变化的活动。[IFRS Foundation：IAS 7](https://www.ifrs.org/issued-standards/list-of-standards/ias-7-statement-of-cash-flows.html)
- **传导**：若经营现金流不足以覆盖经营扩张和必要投资，缺口可能由借款或股权融资填补；这会改变融资现金流、利息/股本要求，进而影响可分配给资本提供者的现金。

## 5. 社会 / 个体

- **具体问题**：消费者的付款时点为何会改变企业扩张所需的资金量？
- **可验证事实 / 来源 [共识]**：客户在企业转移商品或服务前付款时，IFRS 15 要求企业确认合同负债，直至履约。[IFRS 15，第105–106段](https://www.ifrs.org/content/dam/ifrs/publications/pdf-standards/english/2023/issued/part-a/ifrs-15-revenue-from-contracts-with-customers.pdf?bypass=on)
- **传导**：预付款把消费者支付的现金前置，可能降低企业扩张时的外部资金需求；但其能否持续，取决于客户愿付、履约能力及退款/交付安排。

## 完整、可检验的因果链

**[推断]** 需求增长 → 企业为交付增加采购/生产或给予客户账期 → 经营性存货和应收款增加的速度快于经营性应付款与预收款 → 经营现金流相对利润承压 → 若内部现金不足以覆盖扩张所需的经营投入与投资，筹资活动增加 → 可供资本提供者取得的现金及资本成本压力发生变化。

### 尚待验证的环节

1. 新增收入是否确实需要额外存货、预付款或应收款，而非主要来自既有产能/存量资产。
2. 应收、存货、应付及预收中哪些属于经营性项目；需查年报附注与管理层讨论。
3. 营运资本变动是短期季节性波动，还是可持续的结算条件变化。
4. 现金缺口是否由借款/股权填补，以及融资的期限、成本和约束。
5. 预收款是否可持续，及是否存在退款、履约或收入确认风险。

## 练习问题

选一家收入增长的公司：比较最近两期“收入增量、经营现金流增量、经营性应收/存货/预付款/应付/合同负债增量”，再解释现金缺口由谁承担。
